### Structural Improvements

6. **Inconsistent academic tone**: The paraphrased text oscillates between academic and informal/conversational tone. Many sections use second-person ("you"), negation-heavy filler sentences, and colloquial language that is inappropriate for a thesis.

7. **Redundant negation patterns**: Multiple paraphrased sections include unnecessary sentences like "It's not...", "No one can...", "This will not..." that add no information and appear to be AI-detection padding. These should be removed.

8. **Missing cross-references**: Some paraphrased sections lost internal cross-references (e.g., "See Section X" or "as shown in Table Y") that existed in the original.

9. **Reference format consistency**: Some citations were paraphrased or reformatted (e.g., [26] became inconsistently placed). All citations should follow the same [XX] format consistently.
