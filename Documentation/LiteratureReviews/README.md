# Literature Reviews

This folder contains one review per paper from the main [Literature Review](../LITERATURE_REVIEW.md) for the Crypto World Bank (BCOLBD 2025) project.

Each review includes: **Paper Title**, **Link**, **Authors**, **Affiliation**, **Abstract**, **Introduction**, **Motivation**, **Methodology**, **Results and Discussion**, and **Conclusion** (including relevance to Crypto World Bank where applicable).

**Reference PDFs** from `Documentation/References/` that match a reviewed paper are stored in that paper’s subfolder:
- **Paper 10** (SoK DeFi): `paper_10_sok_defi/` — contains review + `01_Werner_SoK_DeFi_2022.pdf`
- **Paper 11** (SHAP): `paper_11_shap_unified_approach/` — contains review + `13_Lundberg_SHAP_NeurIPS_2017.pdf`
- **Paper 17** (ML explainability, BoE): `paper_17_ml_explainability_finance_default_risk/` — contains review + `14_Bracke_ML_Explainability_BoE_2019.pdf`
