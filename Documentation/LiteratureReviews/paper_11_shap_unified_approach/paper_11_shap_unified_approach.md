# Literature Review: Paper 11

**Paper Title:** A Unified Approach to Interpreting Model Predictions (SHAP)
**Paper Link:** https://papers.nips.cc/paper/2017/hash/8a20a8621978632d76c43dfd28b67767-Abstract.html
**Authors:** Scott M. Lundberg, Su-In Lee
**Affiliation:** University of Washington, Seattle, USA

**Abstract:**
Understanding why a model makes a certain prediction can be as crucial as the prediction's accuracy in many applications. This paper presents SHAP (SHapley Additive exPlanations), a unified measure of feature importance that assigns each feature an importance value for a particular prediction. SHAP connects game-theoretic optimal Shapley values with local explanations, identifying a new class of additive feature importance measures and theoretical results showing there is a unique solution in this class with a set of desirable properties. The paper proposes new SHAP value estimation methods and demonstrates their alignment with human intuition through user studies and several previous methods.

**Introduction:**
The growing complexity of machine learning models—from deep neural networks to gradient-boosted tree ensembles—has created a "black box" problem where practitioners cannot understand why a model produces specific outputs for specific inputs. In high-stakes domains like healthcare, criminal justice, and financial lending, this opacity is unacceptable: a doctor needs to understand why a model predicts cancer, a judge needs to know why a model suggests high recidivism risk, and a loan officer needs to explain why an application was denied. Existing explanation methods (LIME, DeepLIFT, layer-wise relevance propagation, Shapley regression values) each have different theoretical properties and can produce contradictory explanations for the same prediction. This inconsistency erodes trust in model explanations.

**Motivation:**
The fundamental question driving this work is: among all possible methods for explaining individual predictions, is there one that is provably optimal? The authors connect the explanation problem to cooperative game theory, where Shapley values represent the unique solution satisfying a set of desirable axioms (efficiency, symmetry, dummy, additivity). If feature attribution can be framed as a cooperative game—where features are "players" and the prediction is the "payoff"—then Shapley values provide the unique theoretically optimal attribution. This framing provides a principled foundation for model explanation, unlike heuristic methods that lack formal guarantees.

**Methodology:**
SHAP defines the explanation model as an additive feature attribution method: g(z') = φ_0 + Σ φ_i · z'_i, where z' ∈ {0,1}^M indicates feature presence/absence and φ_i are Shapley values. The Shapley value for feature i is: φ_i = Σ_{S⊆N\{i}} [|S|!(|N|−|S|−1)!/|N|!] · [f_x(S∪{i}) − f_x(S)], where f_x(S) is the expected model output when only features in S are present. This requires evaluating the model for all 2^N feature subsets, which is computationally intractable for large feature sets. Three efficient estimation methods are proposed: **(1) KernelSHAP** — a model-agnostic method that reformulates Shapley value computation as a weighted linear regression problem, using a specific weighting kernel π_x(z') = (M−1)/[C(M,|z'|)·|z'|·(M−|z'|)] that provably recovers Shapley values; **(2) TreeSHAP** — an exact, polynomial-time algorithm for tree-based models (Random Forest, XGBoost, LightGBM) that exploits tree structure to compute exact Shapley values in O(TLD²) time (T trees, L leaves, D depth) instead of the exponential brute-force approach; **(3) DeepSHAP** — extends DeepLIFT for neural networks by replacing backpropagation rules with Shapley value computations at each layer, preserving local accuracy.

**Results and Discussion:**
SHAP values satisfy three desirable properties that no other method simultaneously achieves: **(1) Local Accuracy** — φ_0 + Σ φ_i = f(x), meaning feature attributions exactly sum to the model's output for that instance, providing a complete decomposition; **(2) Missingness** — if a feature is missing (z'_i = 0), its attribution is zero; **(3) Consistency** — if a model changes so that a feature's marginal contribution increases or stays the same regardless of other features, that feature's SHAP value will not decrease. The paper proves that SHAP is the **unique** method in the class of additive feature attribution methods that satisfies all three properties. Experiments demonstrate that SHAP produces more consistent explanations than LIME across diverse models—when LIME and SHAP disagree, SHAP's explanation better matches human intuition (validated through user studies with 30 participants). TreeSHAP's polynomial-time computation makes it practical for production deployment with ensemble models, processing thousands of explanations per second.

**Conclusion:**
SHAP provides the theoretically unique optimal explanation for individual model predictions, grounded in Shapley values from cooperative game theory. The Crypto World Bank adopts TreeSHAP as its primary explainability engine: for every loan risk assessment, the system generates SHAP feature attributions showing bank approvers exactly which factors (loan amount: −0.15, repayment history: +0.32, wallet age: +0.08, transaction frequency: −0.05) drove the fraud score prediction. This enables transparent, auditable, mathematically-grounded lending decisions that satisfy regulatory explainability requirements.
